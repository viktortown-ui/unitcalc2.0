Данные хранятся локально, никуда не отправляются.


📊 ЧТО ДЕЛАЕТ ПРИЛОЖЕНИЕ
─────────────────────────────────────────────────────────────────────────────
✓ Рассчитывает, сколько остаётся с одной продажи
✓ Показывает точку безубыточности (сколько продаж чтобы "не в минусе")
✓ Считает, сколько нужно продаж для достижения целевого дохода
✓ Анализирует текущую ситуацию (прибыль/убыток)
✓ Визуализирует расходы и доходы графиками
✓ Сохраняет все данные между сессиями
✓ Даёт пресеты (Маркетплейсы/Услуги) для быстрого старта
✓ Экспорт/импорт JSON (перенос настроек/бэкап)
✓ Печать отчёта (PDF через «Печать» в браузере)
✓ Показывает 3 сценария продаж (−30% / база / +30%)


🗂️ СТРУКТУРА ИНТЕРФЕЙСА
─────────────────────────────────────────────────────────────────────────────
Вкладка 1: "Одна продажа"
  - Цена продажи
  - Себестоимость товара/услуги
  - Комиссии (₽ или %)
  - Доставка и упаковка
  - Потери (возвраты/брак)
  - Налоги (3 типа)

Вкладка 2: "Обязательные расходы"
  - Аренда, зарплаты, реклама
  - Сервисы, кредиты, прочее

Вкладка 3: "Цели"
  - Целевой доход
  - Текущие продажи
  - Результаты расчётов

Вкладка 4: "Графики"
  - Распределение доходов
  - Динамика по периодам


🧮 ЛОГИКА РАСЧЁТОВ
─────────────────────────────────────────────────────────────────────────────
1. С одной продажи остаётся:
   Цена - (Закуп + Комиссии + Доставка + Упаковка + Потери + Налоги)

2. Точка безубыточности:
   Обязательные расходы в месяц / Деньги с одной продажи

3. Продажи для цели:
   (Обязательные расходы + Целевой доход) / Деньги с одной продажи
   (с учётом налога на прибыль, если указан)

4. Текущая прибыль:
   (Продажи × Деньги с продажи) - Обязательные расходы - Налог на прибыль


⚙️ УСТАНОВКА CHART.JS (ДЛЯ ГРАФИКОВ)
─────────────────────────────────────────────────────────────────────────────
Вариант 1: Ручная загрузка (простой)
1. Открой браузер и перейди: https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.js
2. Нажми Ctrl+A (выделить всё), затем Ctrl+C (копировать)
3. Открой файл lib/chart.umd.js в блокноте
4. Удали всё содержимое и вставь скопированное (Ctrl+V)
5. Сохрани файл

Вариант 2: Через командную строку (для продвинутых)
1. Открой терминал/командную строку
2. Перейди в папку проекта: cd /path/to/business-calculator
3. Скачай файл: curl -o lib/chart.umd.js https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.js

Вариант 3: Без графиков (если лень)
- Просто не устанавливай Chart.js
- Калькулятор будет работать, но графики покажут ошибку
- Все расчёты продолжат функционировать

📞 ПОДДЕРЖКА
─────────────────────────────────────────────────────────────────────────────
Это автономное приложение. Все расчёты происходят на твоём компьютере.
Никаких серверов, никакой отправки данных - всё локально.

Если что-то не работает:
1. Проверь, что Chart.js установлен
2. Открой консоль браузера (F12 → Console)
3. Проверь, нет ли ошибок
